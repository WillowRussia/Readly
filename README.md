## Project Structure

The application follows a modular structure to ensure separation of concerns:

```text
Readly/
├── Modules/          # Screen modules
├── NetworkManagers/  # API handling & Data fetching
├── Services/         # Business logic & Data persistence
├── Helpers/          # Extensions & Constants
├── Lottie/           # Animation assets
└── Fonts/            # Custom typography
```

## Tech Stack

- **Language:** Swift 5
- **Interface:** SwiftUI
- **Architecture:** MVP + Clean Architecture
- **Networking:** Native URLSession
- **Storage:** Core Data & UserDefaults 
- **Animations:** Lottie
- **Tools:** Xcode 16, Figma

## Installation
- iOS 16.6+
- Xcode 16.1+
